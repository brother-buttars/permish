Access the dev server from a phone or tablet on the same Wi-Fi network. This requires [HTTPS development](/PermissionForm/development/https-dev/) to be set up first.

## Access URL

Use the LAN IP printed by `dev.sh`, for example:

```
https://192.168.3.1
```

The certificate generated by `dev.sh` includes your LAN IP, so HTTPS works out of the box — but only if the device trusts mkcert's root CA.

## Installing the mkcert CA on iOS

### Step 1: Find the root CA file

```bash
ls ~/Library/Application\ Support/mkcert/rootCA.pem
```

### Step 2: Transfer to the device

AirDrop `rootCA.pem` to the iOS device. You can also email it or serve it via a local HTTP server.

### Step 3: Install the profile

On the iOS device:

1. Open **Settings > General > VPN & Device Management**.
2. Tap the downloaded profile.
3. Tap **Install** and enter your passcode.

### Step 4: Enable full trust

On the iOS device:

1. Open **Settings > General > About > Certificate Trust Settings**.
2. Toggle **full trust** for the mkcert root CA.

## Installing the mkcert CA on Android

### Step 1: Transfer the CA file

Transfer `rootCA.pem` to the device via USB, file sharing, or download.

### Step 2: Install the certificate

1. Open **Settings > Security > Encryption & credentials > Install a certificate > CA certificate**.
2. Select the `rootCA.pem` file.
3. Confirm the installation.

:::note
Some Android versions place certificate settings under different menus. Search "Install certificate" in Settings if the path above does not match your device.
:::

After installing the CA, `https://192.168.x.x` loads without certificate warnings.

## Responsive Design Tips

- Use browser DevTools device emulation for quick layout checks, but always test on a real device for touch behavior and performance.
- Signature capture (`SignaturePad` component) requires touch input testing — mouse emulation does not replicate finger pressure and movement accurately.
- Test both portrait and landscape orientations for the form pages.
- The app uses responsive breakpoints. Verify that navigation, modals, and form layouts adapt correctly.
