## What's Included

The desktop app is a self-contained package built with [Tauri](https://tauri.app/). It includes:

- **SvelteKit frontend** — The same web interface, running in a native window.
- **PocketBase sidecar** — Embedded database for local storage. No external database needed.
- **Node.js sidecar** — Handles PDF generation, email, and SMS.
- **Offline-first storage** — All data stays on your machine. Optionally sync to a server.

## Installation

### macOS

1. Download the `.dmg` file.
2. Open it and drag Permish to your Applications folder.
3. On first launch, macOS may warn about an unidentified developer. Right-click the app and select **Open**.

### Windows

1. Download the `.exe` installer.
2. Run the installer and follow the prompts.
3. Windows SmartScreen may show a warning. Click **More info** then **Run anyway**.

### Linux

1. Download the `.AppImage` file.
2. Make it executable: `chmod +x Permish_0.1.0_amd64.AppImage`
3. Run it: `./Permish_0.1.0_amd64.AppImage`

## Auto-Update

The desktop app checks for updates on launch. When a new version is available, you are prompted to download and install it.
