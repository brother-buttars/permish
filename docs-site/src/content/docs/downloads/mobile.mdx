## How It Will Work

The mobile app uses [Tauri Mobile](https://tauri.app/) to wrap the same SvelteKit frontend in a native iOS and Android shell. It includes the same features as the desktop app:

- Offline-first local storage with embedded PocketBase
- PDF generation and signature capture
- Optional sync to a self-hosted server
- Push notifications for form submissions

## Using the Web App on Mobile (Now)

The web app is fully responsive and works on any mobile browser today. If you have a self-hosted instance:

1. Open your Permish URL in Safari (iOS) or Chrome (Android).
2. The interface adapts to your screen size.
3. Signature capture works with touch input.

For local HTTPS development on mobile devices, see [Mobile Testing](/PermissionForm/development/mobile-testing/).
