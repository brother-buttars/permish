Get Permish running in three commands. You need [Docker](https://docs.docker.com/get-docker/) and [Docker Compose](https://docs.docker.com/compose/install/) installed.

## Try It Now

```bash
git clone https://github.com/brother-buttars/permish.git
cd permish
docker compose up -d
```

That's it. Open [http://localhost:3000](http://localhost:3000) in your browser.

## What's Running

| Service   | Port | Description                     |
|-----------|------|---------------------------------|
| Frontend  | 3000 | SvelteKit app (server-rendered) |
| Backend   | 3001 | Express API server with SQLite  |

## First Steps

1. **Register an account** at `/register`. The first user can create events.
2. **Create an event** — add a name, dates, location, and select the youth organizations.
3. **Share the form link** — copy the unique URL and send it to parents.
4. **View submissions** — parents fill out and sign the form. You see completed PDFs in the dashboard.

## Configure Email (Optional)

To send form links via email, create a `.env` file:

```bash
cp .env.example .env
```

Edit `.env` and fill in your SMTP settings:

```ini
JWT_SECRET=your-secret-key-here
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password
EMAIL_FROM=your-email@gmail.com
```

Then restart:

```bash
docker compose down && docker compose up -d
```

:::tip[Gmail App Password]
Use a [Gmail App Password](https://myaccount.google.com/apppasswords), not your account password. Go to Google Account > Security > App passwords.
:::
