Permish is an open-source web application that digitizes the LDS Church **"Parental or Guardian Permission and Medical Release Form."** Event planners create pre-filled events with unique URLs. Parents complete, sign, and submit forms from any device. Signed PDFs are generated by filling in the actual official church PDF.

## Key Features

- **Progressive Web App (PWA)** — Visit the site and install Permish on any device directly from the browser. No app store, no downloads. Works offline with automatic updates.
- **Desktop app** — Native macOS, Windows, and Linux builds via Tauri. Bundles PocketBase for fully offline operation.
- **Offline-first** — Local SQLite storage with optional sync to a server. Submit forms at camp with no cell signal and sync later.
- **Encrypted backup** — Export `.permish-backup` files encrypted with AES-256. Restore on any instance.
- **PDF generation** — Uses `pdf-lib` to fill the actual official church PDF form fields. Drawn or typed signatures are embedded directly.
- **Email and SMS notifications** — Send form links via email or SMS (carrier gateway). Notify planners when forms are submitted.
- **Self-hosted** — Deploy on your own server with Docker Compose. No third-party services required. You own your data.

## How It Works

1. **Event planner** creates an event (e.g., "Young Men Camp 2026") with dates, location, and activities.
2. Permish generates a **unique form URL** for the event.
3. The planner shares the link with **parents and guardians** via email, SMS, or printed QR code.
4. Parents open the link, fill out the form, sign it, and submit.
5. Permish generates a **signed PDF** matching the official church form.
6. The planner views submissions, downloads PDFs, and tracks completion.

## Install as an App (PWA)

Permish is a Progressive Web App. Visit your Permish instance in any browser and install it as a standalone app on your device.

- **iPhone / iPad:** Open Safari, tap Share (arrow up icon), then "Add to Home Screen."
- **Android:** Open Chrome, tap the menu, then "Install app" or "Add to Home screen."
- **Desktop (Chrome/Edge):** Click the install icon in the address bar.
- **Desktop (Safari):** Share, then "Add to Dock."

The installed app works offline, syncs when connected, and receives updates automatically when you open it.
